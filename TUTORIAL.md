## Introduction

This tutorial provides step-by-step instructions for using the AFH Property Advisor application to successfully lease your property to an Adult Family Home provider. Each tutorial builds on the previous one, guiding you through the entire process from initial property analysis to lease execution.

### Prerequisites

Before starting, ensure you have:
- Access to the AFH Property Advisor application
- Property information (address, square footage, room layout)
- Basic understanding of AFH requirements
- Time commitment for the process (30-180 days depending on scenario)

### Tutorial Structure

Each tutorial includes:
- **Objective:** What you'll accomplish
- **Time Required:** Estimated time to complete
- **Prerequisites:** What you need before starting
- **Step-by-Step Instructions:** Detailed guidance
- **Checkpoints:** Verification points
- **Troubleshooting:** Common issues and solutions

## Tutorial 1: Getting Started

### Objective

Set up the application, understand the interface, and review your property information.

### Time Required

30-45 minutes

### Prerequisites

- Property address and basic information
- Access to the application

### Step-by-Step Instructions

**Step 1: Access the Application**

1. Open your web browser
2. Navigate to the application URL (localhost:3000 for development)
3. Wait for the dashboard to load completely
4. Verify you can see the navigation menu at the top

**Step 2: Review the Dashboard**

1. Locate the property information panel
   - Property address should be displayed
   - Verify it matches your property (1713 Winterwood Dr, Centralia, WA)
2. Review the property summary
   - Target capacity: 5-6 beds
   - Property status: Ready for analysis
3. Examine the module cards
   - You should see 6 module cards
   - Each card has an icon, title, and description

**Step 3: Explore Navigation**

1. Click on "Scenarios" in the navigation menu
   - Page should load showing 3 scenarios
   - Return to dashboard using "Dashboard" link
2. Click on "Floor Plan" in the navigation menu
   - Interactive floor plan should be visible
   - Return to dashboard
3. Repeat for other modules to familiarize yourself

**Step 4: Verify Property Information**

1. Check that property address is correct
2. Verify target capacity matches your expectations
3. Review property status
4. Note any information that needs updating

### Checkpoints

- [ ] Application loads successfully
- [ ] Navigation menu is visible and functional
- [ ] Property information is displayed correctly
- [ ] All module pages are accessible
- [ ] You understand the basic layout

### Troubleshooting

**Issue: Application won't load**
- Solution: Check internet connection, clear browser cache, try different browser

**Issue: Navigation not working**
- Solution: Ensure JavaScript is enabled, check browser console for errors

**Issue: Property information incorrect**
- Solution: Note what needs updating, we'll address in later tutorials

## Tutorial 2: Analyzing Your Property

### Objective

Use the Floor Plan Analysis module to understand your property's AFH compliance requirements and estimated conversion costs.

### Time Required

60-90 minutes

### Prerequisites

- Completed Tutorial 1
- Basic understanding of your property layout
- Floor plan or property measurements (helpful but not required)

### Step-by-Step Instructions

**Step 1: Access Floor Plan Module**

1. Click "Floor Plan" in the navigation menu
2. Wait for the floor plan to load
3. Observe the interactive floor plan visualization
4. Note the different colored rooms

**Step 2: Explore Room Information**

1. Click on "Patio" in the floor plan
   - Sidebar should show Patio information
   - Review AFH use, compliance status, modifications needed
   - Note estimated cost: $3,000-$5,000
2. Click on "Primary Bedroom"
   - Review resident room requirements
   - Note modifications: handrails, call system, lighting
   - Note cost: $2,000-$3,000
3. Continue clicking through all rooms:
   - Primary Bathroom
   - Bedroom 1
   - Bedroom 2
   - Bath
   - Living Room
   - Kitchen
   - Dining Room
   - Den
   - Office
   - Laundry
   - Garage
   - CG Room

**Step 3: Review Compliance Analysis**

1. Locate the "AFH Capacity Analysis" panel in the sidebar
2. Review total bed capacity: 5-6 beds possible
3. Examine resident rooms available:
   - Primary Bedroom
   - Bedroom 1
   - Bedroom 2
   - CG Room
4. Review common areas:
   - Living Room
   - Dining Room
   - Den
   - Patio
5. Note bathrooms: 2 (both need modifications)
6. Review compliance score: 75% - Good foundation, needs modifications

**Step 4: Calculate Total Conversion Costs**

1. Review "Conversion Cost Estimates" panel
2. Note minimal WABO compliance: $25,000 - $40,000
3. Note full AFH conversion: $50,000 - $75,000
4. Consider which range fits your budget
5. Document your findings

**Step 5: Identify Critical Modifications**

1. Review bathroom modifications (highest cost items)
   - Primary Bathroom: $5,000-$8,000
   - Bath: $5,000-$8,000
   - Total: $10,000-$16,000
2. Review common area modifications
   - Living Room: $2,000-$3,000
   - Kitchen: $3,000-$5,000
   - Dining: $1,500-$2,500
   - Patio: $3,000-$5,000
3. Review resident room modifications
   - Each room: $1,500-$3,000
   - 4 rooms: $6,000-$12,000
4. Total all modifications to verify estimates

### Checkpoints

- [ ] All rooms have been reviewed
- [ ] You understand compliance requirements for each room
- [ ] You have cost estimates for all modifications
- [ ] You understand total conversion cost range
- [ ] You've identified critical/high-cost items

### Troubleshooting

**Issue: Floor plan not loading**
- Solution: Check internet connection, refresh page, try different browser

**Issue: Room information not displaying**
- Solution: Ensure you're clicking directly on the room rectangle, try clicking different areas

**Issue: Costs seem too high/low**
- Solution: These are estimates based on typical costs, adjust for your local market

## Tutorial 3: Choosing a Scenario

### Objective

Compare all three scenarios, understand trade-offs, and select the best approach for your situation.

### Time Required

90-120 minutes

### Prerequisites

- Completed Tutorial 2
- Understanding of your budget constraints
- Timeline preferences
- Risk tolerance assessment

### Step-by-Step Instructions

**Step 1: Access Scenarios Module**

1. Click "Scenarios" in the navigation menu
2. Review the page layout
3. Observe all three scenarios are displayed
4. Note the priority labels (HIGHEST, RECOMMENDED, OPTIMAL)

**Step 2: Analyze Scenario 1: Direct Provider Lease**

1. Read the complete scenario description
2. Review the PROS section:
   - Fastest path to revenue (30-60 days)
   - No upfront conversion costs
   - Provider handles all permits and modifications
   - Lower risk for landlord
3. Review the CONS section:
   - Limited pool of providers
   - May require longer lease term
   - Lower rent potential initially
4. Examine the timeline: 30-90 days to find provider
5. Review budget: $0 - $5,000 (minor repairs only)
6. Study the implementation steps (6 steps listed)
7. Review marketing approach
8. **Decision Point:** Does this fit your situation?
   - Yes if: Need quick revenue, limited capital, low risk tolerance
   - No if: Want higher rent, have capital, prefer more control

**Step 3: Analyze Scenario 2: Minimal WABO Conversion**

1. Read the complete scenario description
2. Review the PROS section:
   - Property becomes 'turnkey' for providers
   - Higher rent potential ($3,500-$5,000/month)
   - Attracts more qualified providers
   - Faster provider onboarding
3. Review the CONS section:
   - Upfront investment required ($25,000-$40,000)
   - 60-90 day conversion timeline
   - Need to coordinate with contractors
4. Examine the timeline: 90-120 days total
5. Review budget: $25,000 - $40,000
6. Study cost-sharing: Landlord 60-70%, Tenant 30-40%
7. Review implementation steps (6 phases detailed)
8. **Decision Point:** Does this fit your situation?
   - Yes if: Have moderate capital, want balanced approach, prefer turnkey property
   - No if: Need immediate revenue or have very limited capital

**Step 4: Analyze Scenario 3: Full Conversion**

1. Read the complete scenario description
2. Review the PROS section:
   - Maximum rent potential ($5,000-$7,000/month)
   - Attracts premium providers
   - Long-term stable income (10+ year leases)
   - Property value appreciation
3. Review the CONS section:
   - Highest upfront investment ($50,000-$75,000)
   - Longest timeline (120-180 days)
   - More complex project management
4. Examine the timeline: 120-180 days total
5. Review budget: $50,000 - $75,000
6. Study cost-sharing: Landlord 70-80%, Tenant 20-30%
7. Review implementation steps (5 phases detailed)
8. **Decision Point:** Does this fit your situation?
   - Yes if: Have significant capital, long-term investment focus, want premium returns
   - No if: Limited capital or need faster timeline

**Step 5: Compare Scenarios Side-by-Side**

Create a comparison table:

| Factor | Scenario 1 | Scenario 2 | Scenario 3 |
|--------|-----------|------------|------------|
| Initial Investment | $0-$5K | $25K-$40K | $50K-$75K |
| Timeline | 30-90 days | 90-120 days | 120-180 days |
| Monthly Rent | $2.5K-$3.5K | $3.5K-$5K | $5K-$7K |
| Risk Level | Low | Medium | High |
| Provider Pool | Limited | Moderate | Large |
| Best For | Quick revenue | Balanced | Long-term |

**Step 6: Make Your Decision**

1. Consider your financial situation
2. Evaluate your timeline needs
3. Assess your risk tolerance
4. Review your long-term goals
5. Select the scenario that best fits
6. Document your decision and reasoning

### Checkpoints

- [ ] All three scenarios have been thoroughly reviewed
- [ ] Pros and cons of each are understood
- [ ] Timeline and budget for each are clear
- [ ] Comparison table has been created
- [ ] Decision has been made and documented

### Troubleshooting

**Issue: Can't decide between scenarios**
- Solution: Create a pros/cons list for your specific situation, consider consulting with advisor

**Issue: Budget doesn't match any scenario**
- Solution: Consider hybrid approach, adjust cost-sharing percentages, explore financing options

**Issue: Timeline is too long**
- Solution: Consider Scenario 1, or explore ways to accelerate Scenario 2 timeline

## Tutorial 4: Planning Your Timeline

### Objective

Create a detailed timeline with daily activities, costs, and cost-sharing arrangements for your selected scenario.

### Time Required

60-90 minutes

### Prerequisites

- Completed Tutorial 3
- Selected scenario (1, 2, or 3)
- Calendar for scheduling
- Budget information

### Step-by-Step Instructions

**Step 1: Access Timeline Module**

1. Click "Timeline" in the navigation menu
2. Observe the scenario toggle buttons
3. Select your chosen scenario (Minimal or Full)
4. Review the summary information at top

**Step 2: Review Total Timeline**

1. Locate "Total Timeline" in the summary box
2. Note the total days (e.g., 120 days for minimal)
3. Calculate months: divide by 30
4. Mark this period on your calendar
5. Set start date and calculate end date

**Step 3: Review Total Costs**

1. Locate "Total Cost" in the summary box
2. Note the total amount (e.g., $42,500 for minimal)
3. Review "Landlord Share" amount
4. Review "Tenant Share" amount
5. Verify these match your budget

**Step 4: Examine Phase 1 Activities**

1. Locate Phase 1 in the timeline
2. Review each activity within the phase:
   - Days range (e.g., "1-3")
   - Activity description
   - Cost amount
   - Landlord/Tenant percentages
   - Calculated dollar amounts
3. For each activity, note:
   - When it occurs
   - Who is responsible
   - What it costs
   - What needs to be done
4. Mark these dates on your calendar

**Step 5: Continue Through All Phases**

Repeat Step 4 for each phase:
- Phase 2: Contractor Selection
- Phase 3: Construction - Bathrooms
- Phase 4: Construction - Common Areas
- Phase 5: Final Inspections
- Phase 6: Marketing & Provider Search

**Step 6: Create Your Schedule**

1. Create a calendar or spreadsheet
2. Enter all activities with dates
3. Mark responsible parties
4. Note costs and payments due
5. Add buffer time (10-15% extra)
6. Identify critical path items

**Step 7: Plan for Contingencies**

1. Identify activities that could be delayed
2. Plan alternative approaches
3. Set aside contingency budget (10-15%)
4. Identify backup contractors/vendors
5. Plan for inspection delays

**Step 8: Set Up Payment Schedule**

1. Review when payments are due
2. Plan for landlord share payments
3. Coordinate with tenant for their share
4. Set up payment tracking system
5. Plan for cash flow needs

### Checkpoints

- [ ] Timeline has been reviewed phase by phase
- [ ] All activities are scheduled on calendar
- [ ] Costs are understood and budgeted
- [ ] Payment schedule is planned
- [ ] Contingencies are accounted for
- [ ] Critical path is identified

### Troubleshooting

**Issue: Timeline seems too aggressive**
- Solution: Add buffer time, consider extending timeline, prioritize critical items

**Issue: Costs exceed budget**
- Solution: Review cost-sharing, negotiate with contractors, consider phasing work

**Issue: Can't coordinate with tenant**
- Solution: Establish clear communication plan, set regular check-in meetings, use project management tools

## Tutorial 7: Screening Providers

### Objective

Screen potential providers using provided questions and criteria to identify qualified candidates.

### Time Required

30-60 minutes per provider

### Prerequisites

- Completed Tutorial 6
- List of interested providers
- Screening questions ready
- Reference checking process

### Step-by-Step Instructions

**Step 1: Review Screening Criteria**

1. Access Provider Search module
2. Review "Ideal Provider Criteria" section
3. Review "Red Flags" section
4. Understand what to look for
5. Create screening checklist

**Step 2: Initial Screening Call**

1. Schedule phone call with provider
2. Use phone script from Marketing module
3. Ask initial screening questions:
   - How long have you been operating an AFH?
   - How many beds do you currently operate?
   - What is your current occupancy rate?
   - Are you looking to expand or relocate?
4. Take detailed notes
5. Assess initial fit

**Step 3: Conduct Detailed Interview**

1. Schedule longer interview (30-60 minutes)
2. Use all screening questions from module:
   - Experience questions
   - Financial questions
   - Operational questions
   - Timeline questions
   - Modification questions
   - Lease term questions
3. Ask follow-up questions as needed
4. Document all responses

**Step 4: Check References**

1. Request references from provider:
   - Previous landlords
   - Business partners
   - Professional references
2. Contact each reference:
   - Ask about provider's reliability
   - Inquire about business practices
   - Ask about any issues
3. Document reference responses

**Step 5: Verify Licenses and Credentials**

1. Verify AFH license is current
2. Check license status with state
3. Verify business registration
4. Check for any violations or complaints
5. Verify insurance coverage

**Step 6: Evaluate Financial Stability**

1. Request financial information:
   - Credit report (with permission)
   - Bank statements (if appropriate)
   - Business financials
2. Evaluate ability to pay rent
3. Assess financial stability
4. Consider using professional screening service

**Step 7: Conduct Property Tour**

1. Schedule property viewing
2. Observe provider's interest level
3. Answer questions about property
4. Discuss potential modifications
5. Assess provider's vision for property
6. Evaluate compatibility

**Step 8: Score and Rank Providers**

1. Create scoring system:
   - Experience: 25 points
   - Financials: 25 points
   - References: 20 points
   - Fit/Compatibility: 20 points
   - Timeline: 10 points
2. Score each provider
3. Rank providers by score
4. Identify top 2-3 candidates

### Checkpoints

- [ ] All providers have been initially screened
- [ ] Detailed interviews have been conducted
- [ ] References have been checked
- [ ] Licenses have been verified
- [ ] Financial stability has been assessed
- [ ] Property tours have been conducted
- [ ] Providers have been scored and ranked

### Troubleshooting

**Issue: Provider won't provide references**
- Solution: Red flag - consider other candidates, request alternative verification

**Issue: Can't verify financial information**
- Solution: Use professional screening service, request bank letters, consider guarantees

**Issue: Multiple good candidates**
- Solution: Compare offers, consider long-term fit, negotiate with top choice first

## Tutorial 8: Negotiating Lease Terms

### Objective

Negotiate favorable lease terms with selected provider using recommended guidelines.

### Time Required

2-4 weeks (multiple negotiation sessions)

### Prerequisites

- Completed Tutorial 7
- Top provider candidate identified
- Understanding of lease terms recommendations
- Legal advisor available (recommended)

### Step-by-Step Instructions

**Step 1: Review Lease Terms Recommendations**

1. Access Provider Search module
2. Review "Recommended Lease Terms & Conditions" section
3. Understand each term:
   - Lease term (5-10 years preferred)
   - Rent structure (varies by scenario)
   - Cost-sharing arrangement
   - Security deposit
   - Maintenance responsibilities
   - Insurance requirements
4. Determine your priorities
5. Identify negotiable vs. non-negotiable terms

**Step 2: Prepare Initial Offer**

1. Based on your scenario, determine:
   - Base rent amount
   - Lease term length
   - Cost-sharing percentages
   - Security deposit amount
2. Create lease term proposal document
3. Include all key terms
4. Leave room for negotiation
5. Prepare justification for each term

**Step 3: Initial Negotiation Meeting**

1. Schedule meeting with provider
2. Present your initial offer
3. Explain reasoning for each term
4. Listen to provider's response
5. Identify areas of agreement
6. Identify areas needing negotiation
7. Take detailed notes

**Step 4: Negotiate Rent Structure**

1. Discuss base rent amount
2. Consider rent adjustments:
   - Annual increases (CPI-based)
   - Fixed increases
   - Percentage increases
3. Negotiate initial rent amount
4. Agree on adjustment mechanism
5. Document agreement

**Step 5: Negotiate Cost-Sharing**

1. Review conversion costs (if applicable)
2. Discuss cost-sharing percentages:
   - Landlord: 60-80%
   - Tenant: 20-40%
3. Negotiate specific amounts
4. Agree on payment schedule
5. Document agreement

**Step 6: Negotiate Other Terms**

1. Security deposit:
   - Amount (2-3 months rent typical)
   - Return conditions
   - Interest (if applicable)
2. Maintenance:
   - Tenant responsibilities
   - Landlord responsibilities
   - Major repairs
3. Insurance:
   - Tenant requirements
   - Landlord requirements
   - Coverage amounts
4. Subletting and assignment
5. Renewal options

**Step 7: Document Agreements**

1. Create term sheet with all agreements
2. Have both parties review
3. Make revisions as needed
4. Finalize term sheet
5. Both parties sign

**Step 8: Prepare Lease Agreement**

1. Use term sheet to draft lease
2. Include all negotiated terms
3. Add standard lease provisions
4. Have attorney review
5. Revise based on attorney feedback
6. Prepare final draft

### Checkpoints

- [ ] Lease terms have been reviewed
- [ ] Initial offer has been prepared
- [ ] Negotiation meetings have been held
- [ ] All terms have been discussed
- [ ] Agreements have been documented
- [ ] Term sheet has been created
- [ ] Lease agreement is being prepared

### Troubleshooting

**Issue: Provider wants terms you can't accept**
- Solution: Know your limits, be willing to walk away, consider alternative providers

**Issue: Negotiations are stalled**
- Solution: Identify core issues, consider mediator, take break and reconvene

**Issue: Unsure if terms are fair**
- Solution: Consult with attorney, compare to market rates, get second opinion

## Tutorial 9: Executing the Lease

### Objective

Finalize lease agreement, complete legal review, execute documents, and begin lease term.

### Time Required

2-4 weeks

### Prerequisites

- Completed Tutorial 8
- Negotiated terms agreed upon
- Lease agreement drafted
- Legal advisor engaged

### Step-by-Step Instructions

**Step 1: Final Legal Review**

1. Submit lease to attorney for review
2. Address attorney's concerns
3. Make necessary revisions
4. Have provider's attorney review (if they have one)
5. Resolve any legal issues
6. Finalize lease document

**Step 2: Finalize All Details**

1. Verify all terms are correctly stated
2. Ensure all costs are documented
3. Confirm dates are accurate
4. Verify contact information
5. Check all calculations
6. Review entire document

**Step 3: Coordinate Signing**

1. Schedule signing meeting
2. Prepare multiple copies (at least 3)
3. Ensure all parties can attend
4. Have witnesses available (if required)
5. Prepare signing location

**Step 4: Execute Lease Agreement**

1. Review document one final time together
2. Sign all copies
3. Have witnesses sign (if required)
4. Notarize if required by state
5. Exchange copies
6. Keep original in safe place

**Step 5: Collect Security Deposit and First Month Rent**

1. Collect security deposit (per agreement)
2. Collect first month's rent
3. Provide receipts
4. Deposit funds appropriately
5. Document all transactions

**Step 6: Coordinate Property Handover**

1. Schedule property inspection
2. Document property condition
3. Provide keys and access
4. Review property systems
