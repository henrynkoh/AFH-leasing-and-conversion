# AFH Property Advisor - Quick Start Guide

## 5-Minute Quick Start

Get up and running with the AFH Property Advisor in just 5 minutes!

### Step 1: Launch Application (30 seconds)
```bash
cd afh-property-advisor
npm run dev
```
Open http://localhost:3000 in your browser.

### Step 2: Review Dashboard (1 minute)
- Check property information is correct
- Familiarize yourself with the 6 module cards
- Note the navigation menu at top

### Step 3: Quick Scenario Review (2 minutes)
1. Click "Scenarios" in navigation
2. Skim all 3 scenarios
3. Note which one fits your situation:
   - **Scenario 1:** Need quick revenue, no capital
   - **Scenario 2:** Have $25K-$40K, want balance (RECOMMENDED)
   - **Scenario 3:** Have $50K-$75K, want maximum return

### Step 4: Check Floor Plan (1 minute)
1. Click "Floor Plan"
2. Click a few rooms to see details
3. Note total conversion cost estimate
4. Return to dashboard

### Step 5: You're Ready! (30 seconds)
You now understand the basics. For detailed guidance:
- Read TUTORIAL.md for step-by-step instructions
- Read MANUAL.md for comprehensive documentation
- Use QUICKSTART.md for quick reference

## Quick Reference Cards

### Scenario Quick Reference

**Scenario 1: Direct Lease**
- Investment: $0-$5K
- Timeline: 30-90 days
- Rent: $2.5K-$3.5K/month
- Best for: Quick revenue

**Scenario 2: Minimal Conversion** ⭐ RECOMMENDED
- Investment: $25K-$40K
- Timeline: 90-120 days
- Rent: $3.5K-$5K/month
- Best for: Balanced approach

**Scenario 3: Full Conversion**
- Investment: $50K-$75K
- Timeline: 120-180 days
- Rent: $5K-$7K/month
- Best for: Long-term investment

### Cost-Sharing Quick Reference

- **Landlord 100%:** Permits, inspections, design
- **Landlord 70-80%:** Construction, major modifications
- **Tenant 20-30%:** Operational improvements
- **Shared 50/50:** Negotiations, lease finalization

### Timeline Quick Reference

**Scenario 1:**
- Week 1-2: Research providers
- Week 3-6: List property
- Week 7-10: Screen providers
- Week 11-12: Execute lease

**Scenario 2:**
- Days 1-15: Planning & permits
- Days 16-70: Construction
- Days 71-85: Inspections
- Days 86-120: Marketing & provider search

**Scenario 3:**
- Days 1-30: Planning & design
- Days 31-120: Construction
- Days 121-150: Final systems
- Days 151-210: Premium provider search

### Marketing Channels Quick Reference

1. **AFH Locator** - Free, High effectiveness
2. **Facebook Groups** - Free, Very high effectiveness
3. **Direct Outreach** - Time investment, High effectiveness
4. **Commercial Listings** - $100-$500/month, Medium effectiveness

### Provider Screening Quick Reference

**Must Have:**
- ✅ Licensed AFH provider
- ✅ 3+ years experience
- ✅ Strong financials
- ✅ Good references
- ✅ Long-term commitment

**Red Flags:**
- ❌ Poor credit
- ❌ No license
- ❌ Unwilling to commit
- ❌ History of problems
- ❌ No references

## Troubleshooting Quick Fixes

**Application won't load?**
→ Check Node.js version, clear cache, try different browser

**Can't decide on scenario?**
→ Create pros/cons list, consider budget and timeline

**No provider responses?**
→ Improve listing content, try more channels, increase frequency

**Timeline seems too long?**
→ Add buffer time, prioritize critical items, consider Scenario 1

**Costs exceed budget?**
→ Review cost-sharing, negotiate with contractors, phase work

**Provider screening issues?**
→ Use all screening questions, check references thoroughly, verify licenses
