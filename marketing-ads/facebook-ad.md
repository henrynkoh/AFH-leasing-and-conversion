## Best Practices for Facebook Ads

**Image Recommendations:**
- Use high-quality property photos
- Include floor plan image
- Show exterior and interior
- Professional photography preferred
- Include text overlay with key info

**Video Recommendations:**
- 30-60 second property tour
- Show key features
- Include floor plan walkthrough
- Professional narration
- Call-to-action at end

**Ad Placement:**
- Facebook Feed (primary)
- Instagram Feed (if using Instagram)
- Facebook Marketplace
- Facebook Groups (sponsored posts)

**Budget Recommendations:**
- Start with $10-20/day
- Test multiple ad versions
- Run for 7-14 days initially
- Adjust based on performance

**Optimization:**
- A/B test headlines
- Test different images
- Monitor click-through rates
- Adjust targeting based on results
- Pause underperforming ads

**Tracking:**
- Track inquiries from ads
- Monitor cost per lead
- Measure conversion to property viewings
- Track lease signings from ads
